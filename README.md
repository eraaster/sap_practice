# SAP ABAP 실습 과제 모음

본 레포지토리는 SAP ABAP 수업에서 수행한 실습 과제를 정리한 저장소이다.  
각 과제는 주어진 요구사항에 따라 데이터 조회, 가공, 출력(ALV),  
DB 저장 로직을 구현하는 것을 목표로 한다.

## 📂 과제 구성
- assignment_03 : 내부 테이블 가공 및 리포트 출력
- assignment_04 : ALV 기반 사원관리 / 월급지급 시스템

## 🛠 사용 기술
- ABAP Report
- Internal Table / Loop / Group Processing
- ALV Grid
- Selection Screen
- Custom Table (ZEDT19_*)
